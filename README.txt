# Shipping app


##### This project is done by Chadi Kawaf  
The backend is run at the port 8082 and the frontend at 3000


#### Frontend:
npm install
You can run the front end by running the the debug of "scripts" in the package.json file. This will use the http://localhost:3000/
OR
npm run

#### Backend:
Back\src\main\java\com\example\BoxinatorApp, and run the debugger in VSCODE

#### Database 
The database on the port 3306, I used XAMPP as a control Panel, in case the port is not working kill 
the process currently using a port on localhost in Windows and restart XAMPP again.
I used a native MySQL queries as we are not allowed to use anf ORM frameworks.
